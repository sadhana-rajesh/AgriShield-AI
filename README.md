# AgriShield-AI

An AI-powered plant disease detection system that provides real-time health diagnostics and Tamil audio advisories for farmers.

## Features
- Real-time plant disease detection using Deep Learning.
- Instant health diagnostics.
- Tamil audio guidance to assist farmers effectively.

## Technologies Used
- Python
- TensorFlow/Keras
- IoT Integration

## Status
- Currently in development.
## How to Run
1. Clone this repository to your local machine.
2. Open `AgriShield_Main.ipynb` in Jupyter Notebook or Google Colab.
3. Ensure you have the `plant_disease_model.h5` file in the same directory.
4. Run the cells to initialize the system and perform plant disease diagnostics.
